# findOne
## It is a project named "findOne". It can also be understood as find local service providers.

Description:
This is a project named **findOne**. In which a common person who is a user can serach a local service provider nearby his or her. Service providers such as Electrician, Plumber, Mechanic, Carpenter, Painter etc. Who provides their services according to their expertise. This app is really going to solve the real world problem of society. It will revolutionize the way of dealing with service providers and faireness in the work. Such that no person get charged higher amount even if Customer(user) is new and having no any idea of the work and price.
